Python Animal Classes Project

Steps to Build the Project

  1. Create the base class Animal in Animal.py with basic attributes and methods.
  2. Create the Dog class in Dog.py that inherits from Animal and adds dog-specific attributes and methods.
  3. Create the Cat class in Cat.py that inherits from Animal and adds cat-specific attributes and methods.
  4. In main.py, create instances of Dog and Cat using a list of dictionaries containing animal data.
  5. Implement methods to manipulate and display animal data, such as adding tricks/toys, listing them, and updating or deleting animals.
  6. Test the methods by applying them to the instances created and printing the results.

What Students Will Learn

  • How to create and use classes and inheritance in Python.
  • How to manage and manipulate data using dictionaries.
  • How to implement and use instance methods to perform operations on class attributes.
  • How to loop through lists and dictionaries to create and manage instances dynamically.
  • How to update and delete data within a list of dictionaries.
  • How to print and format output to display information clearly.
